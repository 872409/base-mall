## 规范

### 组件

所有组件放置 components 文件夹下且采用“ - ”连接

### 页面

所有页面放置 pages 文件夹下分包除外且采用“ \_ ”连接

### 素材

所有纯色图标采用阿里 icon

### 切片

文件应该遵循的命名规则：
全部小写字母
单词之间使用下划线（\_）连接
不使用缩写作为名称

名称 前缀
Icon ic*
MenuIcon ic_menu*
TabBarIcon ic*tab*
Button btn*
RadioButton btn_radio*
CheckBox btn*check*
Switch btn*switch*
Toggle btn*toggle*

例如
ic_launcher.png
ic_menu_share.png
ic_menu_back.png
ic_menu_settings.png
ic_tab_home.png
ic_tab_mine.png
